This program shows the simulation codes of the research results about Affine Frequency Division Multiplexing (AFDM) waveform
<br>UWB-LAB, School of Information Science and Technology, Harbin Institute of Technology (Shenzhen)

All the codes in this program have been verified to run on 2023b.

File lists:
1. Simu_OS_PS_AFDM.m
<br>These are the simulation codes of the proposed overlap-summation-based pulse shaping transceiver for AFDM systems (OS-PS-AFDM), described in:
Haojian Zhang, etc., ``Overlap-Summation-Based Pulse Shaping Transceiver for Affine Frequency Division Multiplexing'', submitted to Science China Information Sciences, 2026.
